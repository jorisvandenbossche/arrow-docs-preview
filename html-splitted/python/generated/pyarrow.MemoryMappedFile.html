pyarrow.MemoryMappedFile¶

class pyarrow.MemoryMappedFile¶

Bases: pyarrow.lib.NativeFile

A stream that represents a memory-mapped file.

Supports ‘r’, ‘r+’, ‘w’ modes.

  • wb: binary write

  • rb+: binary read and write
